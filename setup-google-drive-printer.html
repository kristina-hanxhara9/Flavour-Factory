🎉 LAJM I MIRË: Google Drive është 100% FALAS deri në 15GB! Mjafton për 10+ vite të dhëna!

Konfiguro në 3 Minuta:

Hapi 1: Hap Google Cloud Console

Kliko Këtu për Google Cloud

Logohu me llogarinë tënde Gmail

Hapi 2: Krijo Projekt të Ri (FALAS)

1. Kliko "Select a project" → "New Project"

2. Emri: bakery-scanner

3. Kliko "Create"

Hapi 3: Aktivizo Google Drive API

1. Kliko "Enable APIs and Services"

2. Kërko: Google Drive API

3. Kliko mbi të → "Enable"

Hapi 4: Krijo Kredencialet (Client ID)

1. Shko te "Credentials" → "Create Credentials" → "OAuth client ID"

2. Application type: Web application

3. Name: Bakery Scanner

4. Authorized JavaScript origins:

http://localhost:8000
https://YOUR-USERNAME.github.io

5. Kliko "Create"

6. Kopjo këto:

Client ID: [do të shfaqet këtu]
API Key: [krijo nga "Create Credentials" → "API Key"]

Hapi 5: Vendosi në Aplikacion

Hap file config.js dhe vendos:

GOOGLE_CLIENT_ID: 'vendos-client-id-këtu',
GOOGLE_API_KEY: 'vendos-api-key-këtu'
✅ GATI! Tani mund të bësh backup automatik në Google Drive!

🖨️ LIDHJA E PRINTER BLUETOOTH

🖨️

Printerët e Rekomanduar (Termik):

  • MUNBYN - 3,000-5,000 Lek
  • RONGTA - 4,000-6,000 Lek
  • Star Micronics - 8,000-12,000 Lek

Si të Lidhësh:

Në Telefon/Tablet:

1. Ndiz printer dhe Bluetooth

2. Hap Settings → Bluetooth

3. Gjej printer (p.sh. "MUNBYN-001")

4. Kliko për të lidhur

5. Hap aplikacionin → Settings → Connect Printer

Në Kompjuter:

1. Duhet Chrome ose Edge browser

2. Sigurohu që je në HTTPS (jo HTTP)

3. Kliko "Lidh Printer Bluetooth"

4. Zgjidh printer nga lista

Kujdes: Bluetooth funksionon VETËM në:
  • HTTPS (jo HTTP)
  • Chrome ose Edge browser
  • Android/Windows (iOS ka kufizime)

❌ ZGJIDHJA E PROBLEMEVE

Problem: "Permission dismissed" për kamerën

Zgjidhje:

1. Kliko ikonën e "lakut" pranë URL

2. Gjej "Camera" → Ndrysho në "Allow"

3. Rifresko faqen (F5)

Problem: Ikonat nuk shfaqen (404)

Zgjidhje e shpejtë:

Problem: Google Drive nuk lidhet

Kontrollo që ke vendosur Client ID dhe API Key në config.js

Sigurohu që je në HTTPS ose localhost
